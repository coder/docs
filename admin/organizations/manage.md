This article will show you how to:

- Create a new organization
- Delete an organization
- Add or remove members to an organization

## Creating an Organization

Site managers can create new organizations by going to **Manage** >
**Organizations**.

Select **New Organization** in the top-right. When prompted, provide a **name**
and a **description** for the new organization.

If you want to set this as the default organization, check the **default
organization** box.

## Deleting an Organization

You can delete an organization by going to **Manage** > **Organizations**. Find
the organization you're interested in and select it. In the top-right, click
**Delete**.

Please note that you cannot delete organizations to which active environments
are assigned.

<video autoplay loop="loop" width="100%" controls
src="../../../assets/delete-org.mp4"></video>

## Adding / Deleting Organization Members

You can add or delete members to an organization by going to **Manage** >
**Organizations**.

Select the organization with which you want to work and click **Edit Members**.
Find the user you wish to add or remove.

To add a user, check the box next to their name. To remove a user, uncheck the
box next to their name.
