Coder moves software development to the cloud, delivering consistent, secure,
and performant environments to development teams.

If you're interested in:

- Bringing remote development environments to your team, see [Why
  Coder?](about/why-coder.md)
- Getting started with your Coder auto-generated license, see [our installation
  instructions](../admin/install.md)  
- Setting up remote development environments by yourself, see
  [code-server](https://github.com/cdr/code-server) and
  [sshcode](https://github.com/cdr/sshcode)
